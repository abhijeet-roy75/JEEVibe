# Logo Assets

Place your JEEVibe logo image here.

## File Name
- `jeevibe_logo.png`

## Recommended Specifications
- **Format**: PNG (with transparency preferred)
- **Size**: 400x400px or larger (will be scaled down)
- **Aspect Ratio**: Square (1:1) or close to it
- **Background**: Transparent or white

The logo will be displayed on the home screen. If the logo file is not found, the app will fall back to a camera icon.
