# Enable GitHub Pages for Privacy Policy

## Quick Steps

1. **Go to your GitHub repository**:
   - https://github.com/abhijeet-roy75/JEEVibe

2. **Settings → Pages**:
   - Click **Settings** tab
   - Scroll down to **Pages** in left sidebar

3. **Configure Source**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
   - Click **Save**

4. **Wait 1-2 minutes** for deployment

5. **Your Privacy Policy URL will be**:
   ```
   https://abhijeet-roy75.github.io/JEEVibe/PRIVACY_POLICY
   ```

## Use This URL in Google Play Console

1. Go to [Play Console](https://play.google.com/console)
2. Select **JEEVibe**
3. Navigate to: **Policy** → **App content** → **Privacy Policy**
4. Enter the URL
5. Click **Save**

---

**Note**: The raw GitHub URL also works if you prefer not to set up Pages.
